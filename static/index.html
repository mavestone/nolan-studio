Welcome to Nolan

Local-first documentary studio. Whisper transcribes your footage, scene detection tags every shot, and an AI helps you find the gold.

This setup takes about 60 seconds. You can change everything later from Settings.

AI keys (optional)

Transcription + scene detection run fully offline. AI keys only matter for chat, story analysis, and intelligent search. Paste any you have — skip the rest.

Telegram bot (optional)

Chat with your footage from your phone. Create a bot via @BotFather → /newbot → paste the token below.

After saving, send /start to your bot from Telegram — it'll reply with your chat ID, which you'll paste into Settings to allowlist yourself.

Your first project

A project is a film. Inside, you add bins (folders) and scan them to discover clips. Or import a .nolanproj bundle from a collaborator if they've already processed.
